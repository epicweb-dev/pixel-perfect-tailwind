## Composing font-size and line-height

ğŸ‘¨â€ğŸ’¼ As you know, the way we've extended font sizes in the Tailwind config defines both a `font-size` and a `line-height` for each size. Make sure we support both these in the CSS config too!

ğŸ’° To compose a `line-height` with a `font-size`, you can create a `--` _modifier_ CSS variable for a given font size.

For example:

```css
@theme {
	--font-size-woof: 1.25rem;
	--font-size-woof--line-height: 1.5;
}
```

The `--line-height` appended to the `woof` font size variable will compose that line-height with the font size in the generated CSS.
